Project Intent

This dashboard is designed as a Persistent Ambient Display. The goal is to bridge the gap between a functional timepiece and a personal gallery, serving as an "Always-On" companion for a professional workspace.

Technical Architecture

  • Screen Wake Lock API: Prevents the host device from entering sleep mode or dimming the display, ensuring the clock remains visible during long work sessions.
  • Fisher-Yates Shuffle: An $O(n)$ complexity algorithm used to ensure an unbiased, non-repeating random sequence for the content rotation.
  • Deterministic Minute-Reset: State-management logic that synchronizes UI transitions precisely with the 00-second mark of the system clock.
  • Hybrid Background Engine: Supports both Synchronized (mirrored) and Independent background cycling with a deterministic state-management loop.
  • Stealth Gesture Layer: Implements a "Ghost UI" using double-click event listeners and Keyboard Hotkeys to toggle state without persistent UI clutter.
